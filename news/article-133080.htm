第二章 实战安装：跨平台部署详解

2.1 Windows系统安装全流程

  1. 获取安装包：

  2. 安装注意事项：

第三章 深度配置：打造专属代理方案

3.1 节点配置的艺术

获取节点信息后，在轻舟界面点击"服务器→添加"，需要注意以下高级参数：

| 参数项 | 推荐设置 | 技术说明 | |--------------|---------------------------|------------------------------| | 传输协议 | WS(WebSocket)或mKCP | 前者兼容性好，后者抗干扰强 | | TLS类型 | 推荐1.3版本 | 避免使用已弃用的TLS1.0/1.1 | | 伪装类型 | 根据网络环境动态选择 | 校园网建议http伪装 | | 流控策略 | 高延迟网络启用BBR算法 | 显著提升视频流畅度 |

3.2 路由规则定制

通过"路由设置"可以实现：
- 分应用代理：只让浏览器走代理，游戏客户端直连
- 智能分流：国内网站直连，境外流量代理
- 广告拦截：整合Hosts规则屏蔽广告域名

示例规则片段：
json "routing": { "domainStrategy": "IPIfNonMatch", "rules": [ { "type": "field", "outboundTag": "direct", "domain": ["geosite:cn"] } ] }

4.2 常见问题解决方案

Q1 连接成功但无法访问
→ 检查系统时间误差需小于1分钟
→ 尝试切换传输协议
